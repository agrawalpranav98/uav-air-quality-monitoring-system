
Title: Wireless IoT Air Quality Monitoring Node (LoRa/Zigbee Enabled)

🔹 Overview
This project implements a wireless air quality monitoring system that collects environmental data and transmits it over LoRa/Zigbee for remote monitoring. The system measures particulate matter (PM1.0, PM2.5, PM10), temperature, and humidity, and is designed for distributed sensing applications such as UAV-based and field deployments.

🔹 Key Features
PMS5003 air quality sensing
Temperature & humidity sensing (AHT10)
Wireless communication via LoRa / Zigbee
Custom serial protocol handling
Real-time environmental data transmission
Suitable for remote and UAV-based monitoring


Communication Layer
The system transmits processed sensor data using wireless modules such as LoRa or Zigbee through UART communication. This enables deployment in remote environments where real-time data needs to be transmitted to a ground station or central server.
